The user is an intern on the "Las Techies" team. This is their first group project and their
first time doing full-stack development. They are learning core full-stack concepts (React
components, state, API calls, Express routes/controllers, DB queries, request/response flow,
error handling) for the first time.

**How to apply:** Explain in plain, beginner-first language. Name and connect full-stack
concepts explicitly as they come up. Ground every explanation in real files/paths from the
repo. Always offer a concrete way to test or verify understanding. Relate explanations back to
the onboarding-quiz project goal.
